# Running Performance Factor Analysis (R)

## Overview
Statistical analysis of women's national running records using factor analysis to identify latent performance factors.

## Objectives
- Validate dataset suitability for factor analysis
- Reduce dimensionality of performance variables
- Identify underlying athletic performance factors

## Methods
- Data validation and preprocessing
- Mahalanobis distance outlier detection
- KMO and Bartlett tests
- Factor analysis with Varimax and Oblimin rotations

## Results
Three latent factors were identified:
- Speed/Short-distance endurance
- Middle-distance endurance
- Long-distance endurance

## Technologies
- R
- Multivariate statistics
- Factor analysis
